How to use:

  1. Hit the "Generate new run" button
  2. Download the zip file on the next page (you can share this download page url if you'd like to share this run seed)
  3. Unzip
  4. Apply the ips file to an ALREADY PATCHED CK+ rom
    • I'd recommend https://www.marcrobledo.com/RomPatcher.js/ to patch
    • This randomizer was compiled for version md5:9512a4fa13f8398eeded97f9e240d1a0 of CK+
  5. Install the browser extension to load the json file into Emi calc
